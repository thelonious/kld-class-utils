kld-class-utils
===============

A collection of JS "class" utilities I've collected and/or written over time. Nothing amazing here, but they work for me.

Installation
============
    npm install kld-class-utils


Utilities
=========
* isObject(o)
* isString(s)
* isMethod(o, m)
* extend(obj, props)
* inherit(proto)
* defineSubclass(superclass, constructor, methods, statics)
